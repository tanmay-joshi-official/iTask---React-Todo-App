# âš™ï¸ How It Works

- Todo text is stored using useState  
- Todo list is managed as an array of objects  
- Each todo has id, text, and completion status  
- useEffect loads todos from localStorage on first render  
- Another useEffect saves todos whenever list updates  
- useRef prevents overwriting localStorage on first render  
- Array methods handle edit, delete, and toggle logic  
- Tailwind handles layout and responsiveness
